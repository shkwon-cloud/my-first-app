# 꿈해몽 AI 로또 추천

어젯밤 당신의 꿈을 행운의 번호로 바꿔보세요.

## 바로가기
- **배포 사이트**: [https://my-first-app-cm5.pages.dev/](https://my-first-app-cm5.pages.dev/)
- **GitHub 저장소**: [https://github.com/shkwon-cloud/my-first-app](https://github.com/shkwon-cloud/my-first-app)

## 기능
- 꿈 내용 입력 및 분석 시뮬레이션
- 분석 기반 행운의 로또 번호 생성
- 몽환적인 밤하늘 디자인

## 기술 스택
- HTML5, CSS3, JavaScript (Vanilla JS)
- Cloudflare Pages (Deployment)
